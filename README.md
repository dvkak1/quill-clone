# Quill Website Clone (In Progress)

## 📌 Overview

This project is a **course-based implementation** of the Quill website, recreated as part of my front-end development training using the Future Fullstack CSS course.

The goal of this project is to **apply and reinforce core HTML and CSS concepts** through a structured, hands-on build. It focuses on translating design into code while following modern front-end practices.

---

## 🎯 Objectives

* Practice **semantic HTML structure**
* Apply **CSS fundamentals** (text styling, spacing, layout)
* Implement a consistent **typography system (type scale)**
* Understand and use **CSS variables**
* Build layouts using **Flexbox**
* Improve **code organization and readability**

---

## 🛠️ Technologies Used

* HTML5 (Semantic Markup)
* CSS3
* Flexbox
* CSS Variables (`:root`)
* Responsive Design Principles (in progress)

---

## 🚧 Current Status

This project is **still in development** as I progress through the course.

Completed so far:

* Basic page structure (HTML)
* Navigation layout
* Typography system (font sizes, weights, spacing)
* Initial styling using CSS variables
* Flexbox implementation for layout sections

In progress:

* Refining layout and spacing
* Improving responsiveness
